# Customizing Your Valentine's Website

Welcome to your personalized Valentine's website! Follow the instructions below to customize your site with personal photos, messages, and music.

## Step 1: Adding Photos
1. Navigate to the `assets/images` directory in your project.
2. Upload your personal photos here. Make sure the images are in JPG or PNG format.
3. Update the HTML file to reference your images. For example:
   ```html
   <img src="assets/images/your-photo.jpg" alt="Your Photo">
   ```

## Step 2: Customizing Messages
1. Open the `index.html` file in your root directory.
2. Find the section where messages are displayed. For example:
   ```html
   <p>Happy Valentine's Day!</p>
   ```
3. Replace the text with your own personalized message.

## Step 3: Adding Music
1. Place your music files (MP3 format) in the `assets/music` directory.
2. Modify the HTML file to include your music file. For example:
   ```html
   <audio controls>
     <source src="assets/music/your-song.mp3" type="audio/mpeg">
     Your browser does not support the audio element.
   </audio>
   ```

## Final Touches
- Make sure to test your site in different browsers to ensure compatibility.
- Enjoy your customized Valentine's website!
